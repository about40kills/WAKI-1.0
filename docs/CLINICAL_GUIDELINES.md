# Clinical Guidelines

## Four Non-Negotiable Rules

1. Never diagnose.
2. Never prescribe.
3. Always escalate crisis at high risk.
4. Never delay referral when user asks for professional help.

## Safeguard Statement

WAKI is not a therapist, doctor, or crisis service. It supports wellbeing and routes users to human help when needed.

Twi version:

## Ethical Framing

- Coping tools are self-guided relaxation techniques, not treatment.
- They are optional and never prescribed.
- Persistent or severe distress always receives referral guidance.
